Ansible role FreeBSD Poudriere
##############################

**vbotka.freebsd_poudriere**

Role version 2.6.3

This `role <https://galaxy.ansible.com/vbotka/freebsd_poudriere/>`_ and the documentation is work in
progress. The current state of this document describes successful configuration and building of
amd64 and armv7 packages. Other architectures should work out of the box.
